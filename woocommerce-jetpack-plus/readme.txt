== Description ==

WooCommerce Jetpack is a WordPress plugin that supercharges your site with awesome powerful features. Features are absolutely required for anyone using excellent WooCommerce platform.
Unlock all WooCommerce Jetpack features and supercharge your WordPress WooCommerce site even more.

= Unlocked features =
* Custom Price Labels – Additional options when creating custom price labels for products.
* Call for Price – Default text editing.
* Currencies – Currency symbol editing.
* Sorting – Remove all sorting option.
* Order Numbers – Order number prefix and width enabled.
* PDF Invoices – Invoices for customers enabled.
* Product Info – Priority (i.e. Order) enabled.
* Emails – Bcc and Cc emails enabled.
* Orders – Error messages editing enabled.
* Payment Gateways - Minimum cart amount editing enabled.

== Installation ==

1. Upload the entire 'woocommerce-jetpack-plus' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce > Settings > Jetpack.

== Frequently Asked Questions ==

= Should I leave WooCommerce Jetpack plugin enabled and activated when I install WooCommerce Jetpack Plus? =

Yes, WooCommerce Jetpack plugin MUST be installed and activated when you install WooCommerce Jetpack Plus.
